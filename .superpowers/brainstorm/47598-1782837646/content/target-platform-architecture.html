推荐设计：平台骨架 + AI 投研引擎

不是简单重写，也不是直接 fork。Money_Never_sleep 保持自己的平台边界，TradingAgents-astock 作为深度 AI 分析内核接入。

目标分层架构
体验层
Web 工作台 / 桌面客户端 / Agent 对话 / 报告阅读 / 自选股与组合看板
平台 API 层
FastAPI 服务、任务队列、历史报告、配置、用户组合、分析会话、权限与审计
AI 投研层
快速问股 Agent / 深度多 Agent 投研图 / 风控辩论 / 决策评分 / 策略技能
数据与知识层
A 股行情、财务、新闻、公告、资金流、龙虎榜、解禁、缓存、标准化、记忆
验证与风控层
回测、信号归因、置信度、止损纪律、成本预算、可追溯日志、投资免责声明
从三个项目吸收什么

TradingAgents-astock：7 分析师、Bull/Bear 辩论、风险辩论、Portfolio Manager。

daily_stock_analysis：API/Web/桌面边界、任务、报告、通知、数据 fallback、测试和 CI。

go-stock：Wails 桌面经验、行情图表、工具分组、ReAct / PlanExecute 模式、MCP 工具想法。

第一阶段不做什么

不承诺自动赚钱，不做黑箱荐股。

不直接复制三个项目的大量代码。

不一开始同时做 Web、桌面、回测、选股、通知全家桶。

先做一个可解释、可追溯、可复盘的单股深度分析闭环。
